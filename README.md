# Wedding Invitation Site

Αρχεία για το one-page wedding invitation site σε GitHub Pages.

## Περιεχόμενα
- `index.html` → η βασική σελίδα
- `404.html` → προαιρετική ανακατεύθυνση
- `.nojekyll` → βοηθά το GitHub Pages να σερβίρει τα αρχεία όπως είναι
- `assets/images/watercolor-purple.png` → background της πρώτης οθόνης

## Ανέβασμα στο GitHub
1. Ανέβασε όλα τα αρχεία στο root του repository.
2. Πήγαινε `Settings > Pages`.
3. Επίλεξε `Deploy from a branch`.
4. Διάλεξε branch `main` και folder `/(root)`.
5. Save.
